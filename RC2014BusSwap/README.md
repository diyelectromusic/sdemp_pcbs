# RC2014 Bus Swap

This is a converter to swap the pin directions/orientations of an RC2014 bus.

- Design and build notes: https://emalliab.wordpress.com/2026/05/12/rc2014-bus-reverser/

Bill of Materials:
- RC2014 Bus Swap PCB.
- 2x 40-way right angle pin header sockets.
- Optional: 100nF ceramic capacitor.
- Optional: 2-way right angle pin header and jumper
