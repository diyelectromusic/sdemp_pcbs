# Pimoroni PGA2350 Breakout

This ia simple breakout board for the Pimoroni PGA2350 module.

- Design notes: https://diyelectromusic.com/2026/05/04/pga2350-breakout-pcb-design/
- Build and test notes: https://diyelectromusic.com/2026/05/04/pga2350-breakout-pcb-build-guide/

Bill of Materials:
- PGA23505 Breakout PCB.
- Pimoroni PGA2350.
- SMT micro USB socket (see photos and PCB for footprint).
- 2x 2-pin tactile button switches.
- Double row, round pin header sockets.
- Double row, extended round pin header pins.
- Pin header pins or sockets as required for the breakout.

Errata:
- The holes for the buttons are too small (for me).
